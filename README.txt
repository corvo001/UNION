UNION v0.2
UNION es un ecosistema modular para la generación, procesamiento y visualización de fractales con inteligencia artificial integrada.
La versión 0.2 introduce mejoras clave en la comunicación entre módulos y estabilidad general del flujo de datos.

Novedades en la 0.2
Integración estable Creator → Raven → Nexo → FractalExplorer: el flujo de capturas, renderizado y retroalimentación visual ya está funcional.

Sistema de Output compartido: las recreaciones procesadas por Raven se almacenan en la carpeta shared del renderizador visual para interpretación inmediata.

Optimización de transferencia: reducción de latencia en la comunicación entre módulos y manejo más eficiente de archivos intermedios.

Estructura modular refinada: cada componente mantiene independencia, pero sigue un protocolo de comunicación unificado.

Estado actual
La 0.2 establece la base sólida del ecosistema, priorizando la fiabilidad del flujo de datos.
En la siguiente versión se expandirá la capacidad de Raven para generar mutaciones visuales más complejas y añadir control de interacción en tiempo real.
